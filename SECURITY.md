# Security Policy

## Overview

USABench is an open research project made available for the public benefit.
It is provided **as is**, without warranty or commitment to maintenance or
ongoing support.

The maintainers and project sponsors **do not provide security updates,
patches, or vulnerability response services** for this repository.

## Responsible Use

Users are encouraged to review the code, data, and evaluation framework
for their own security and compliance needs before deploying or modifying
this software in any environment.

Contributors should avoid submitting or including any data, code, or
information that is confidential, proprietary, or personally identifiable.
